# Gif-Getter

#### Overview

This app uses Jquery to make AJAX calls to the Giphy API.  Jquery is also used to dynamically create buttons based on user inputs and to pause and unpause the gifs.  

#### Direcory Layout

```
Giphy-Page
    - assets  
        - css
            - reset.css
            -style.css
        - javascript
            - js.js
    - Index.html
```
